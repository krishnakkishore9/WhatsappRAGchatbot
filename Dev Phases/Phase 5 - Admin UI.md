# Phase 5: Simple Admin UI

## 🎯 Objective
Create a functional, minimal web interface for administrators to manage documents and view logs.

## 📝 Tasks
- [ ] Set up FastAPI static files serving.
- [ ] Create `index.html`:
    - Simple file upload form.
    - Document status list (refreshable).
    - Basic chat log viewer.
- [ ] Use minimal CSS (or Bootstrap CDN) for basic layout.
- [ ] Implement Vanilla JS to call backend endpoints (`/admin/upload`, `/admin/documents`, `/admin/logs`).

## ✅ Success Criteria
- [ ] Admin can upload files through the browser.
- [ ] Admin can see the list of indexed documents.
- [ ] Recent WhatsApp interactions are visible in the UI.
